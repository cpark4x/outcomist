Why this matters
Different questions need different approaches. "Should I?" explores patterns and deeper motivations. "How do I?" focuses on practical execution.
Evaluation

Pros

  • Self-explanatory with examples
  • User can identify their situation
  • Clear differentiation between paths
  • No prerequisite reading required

Cons

  • Loses the reframing/insight
  • Doesn't demonstrate Outcomist's value
  • User might not see the pattern
  • Feels more generic/transactional
